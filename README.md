# Movie Recommender System

A content-based movie recommender system built with Python and Streamlit. The app recommends similar movies based on movie tags and displays posters using the TMDB API.

## Live Demo

[Open the deployed app](https://movie-recommender-system-2mef.onrender.com)

## Features

- Search or select a movie from the dropdown
- Get top 5 similar movie recommendations
- View movie posters fetched from TMDB
- Content-based recommendation using cosine similarity
- Deployed on Render

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Requests
- TMDB API
- Render
